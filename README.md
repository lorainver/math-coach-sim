# 🧮 MO 教练模拟器 (Math Coach Simulator)

一款基于命令行的数学竞赛教练模拟游戏。你将扮演中学数学竞赛主教练，选择学生、做题提升能力，目标是将所有学生的代数、几何、数论、组合四项能力全部提升至"优"评级——达成 **AKIMO**！

## 🎮 游戏介绍

> 时光荏苒，白云苍狗。不知不觉中，你教的学生已到了高一。而你，作为 xx 中学的数学竞赛主教练，将指导学生参加比赛，并争取获得成绩。

**游戏目标：协助学生 AKIMO（All Kill IMO）。**

### 玩法概述

1. **选择学生**：从四位学生中挑选 2 名（甲子、乙丑、丙寅、丁卯）
2. **做题提升**：选择代数 / 几何 / 数论 / 组合四个板块，系统随机分配难度
3. **答题判定**：答对提升对应能力，答错则无变化
4. **达成 AKIMO**：当两名学生的四项能力全部达到"优"评级时，游戏胜利！

### 四项能力指标

| 板块 | 说明 |
|------|------|
| **代数** (Algebra) | 方程求解、函数分析、不等式 |
| **几何** (Geometry) | 三角形、四边形、立体几何 |
| **数论** (Number Theory) | 整除、同余、因式分解 |
| **组合** (Combinatorics) | 计数、排列组合、构造 |

### 评级体系

能力从低到高分为四个等级：**差 → 中 → 良 → 优**

### 难度分级

| 难度 | 说明 | 能力提升 |
|------|------|----------|
| ⭐ 简单 | 基础入门题 | +1 |
| ⭐⭐ 中等 | 竞赛常规题 | +1 |
| ⭐⭐⭐ 困难 | 高联 / CMO 真题 | +2 |

### 学生一览

| 学生 | 代数 | 几何 | 数论 | 组合 |
|------|------|------|------|------|
| **甲子** | 差 | 中 | 中 | 良 |
| **乙丑** | 中 | 差 | 中 | 差 |
| **丙寅** | 差 | 差 | 中 | 差 |
| **丁卯** | 差 | 良 | 差 | 差 |

### 题库来源

题目涵盖从基础数学到竞赛高阶：
- 基础题：简单方程、三角形内角和等
- 竞赛题：全国高中数学联赛（高联）历年真题
- 高阶题：中国数学奥林匹克（CMO）试题
  - 1997 / 2024 / 2025 高联一试
  - 2025 CMO

## 🖥️ 编译与运行

### 环境要求

- C++ 编译器（支持 C++11 及以上）
- Windows 终端（推荐使用支持 Unicode 的终端以正确显示中文）

### 编译

```bash
g++ math_coach_sim.cpp -o math_coach_sim -finput-charset=UTF-8
```

### 运行

```bash
./math_coach_sim
```

## 🛠️ 技术细节

- **语言**：C++
